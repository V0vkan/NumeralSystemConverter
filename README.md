# NumeralSystemConverter

The program takes three lines:

1. The source radix;
2. The source number;
3. The target radix;

Then, it will output converted number to target radix

# Example

Input:

10

11

2

Output:

1011


# Fractional numbers can also be converted from one base to another.

# Examples
